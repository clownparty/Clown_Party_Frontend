PokéTeam Builder

PokéTeam Builder allows you to build out, edit, and, if desired, delete teams of six pokémon. You can also view those created by other users.

In the future, we would like to include types, user bios, and a random team display (in which a random team pulled from the database would be displayed). A random team generator (in which six random pokémon would be placed in a team) would also be a good addition. Eventually, the ability to give your pokémon moves would be a good addition.
